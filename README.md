# Quatermaster

The "Solitude Of War" Discord Bot


```
usage: quartermaster.py [-h] [-t TOKEN | -f TOKEN_FILE] [-v VERBOSITY]
                        [-l LOG_FILE] [-vv LOG_FILE_VERBOSITY]

The "Solitude Of War" Discord Bot

optional arguments:
  -h, --help            show this help message and exit
  -t TOKEN, --token TOKEN
                        API Token
  -f TOKEN_FILE, --token-file TOKEN_FILE
                        File which contains API Token; default: api.key

logging:
  There are various levels of logging, in order of verbosity: critical,
  error, warning, info, debug, noset.

  -v VERBOSITY, --verbosity VERBOSITY
                        Set verbosity for console output; default: error
  -l LOG_FILE, --log-file LOG_FILE
                        File to log bot status. Not used by default.
  -vv LOG_FILE_VERBOSITY, --log-file-verbosity LOG_FILE_VERBOSITY
                        Set log file verbosity; default: debug

```
